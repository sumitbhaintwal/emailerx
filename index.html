This is the Header

Use .header class on table to create header.

For eg.

<table class="header">
<tr>
<td>
<p>This is the Header example</p>
</td>
</tr>
</table>

You can modify background, paddings and all in CSS(obviously).

All the low width section with white background is contained in .container class.

The markup is very simple.

<table class="container">
<tr>
lt;td>
<p>This is a container</p>
</td>
</tr>
</table>

The above is an example of one-columned container. There is no special class for creating one-columned container. It is a default thingy.
